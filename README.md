# Strategi Algoritmik Praktikum 1–10

Repository ini berisi implementasi berbagai algoritma yang dipelajari pada mata kuliah Strategi Algoritmik menggunakan bahasa C++.

## Daftar Praktikum

### Praktikum 1 - Analisis Algoritma
Mempelajari konsep dasar analisis algoritma, notasi Big O, dan efisiensi program.

### Praktikum 2 - Brute Force
Implementasi algoritma brute force untuk menyelesaikan permasalahan pencarian dan optimasi sederhana.

### Praktikum 3 - Divide and Conquer
Implementasi strategi divide and conquer untuk memecah masalah menjadi beberapa submasalah yang lebih kecil.

### Praktikum 4 - Greedy Algorithm
Implementasi algoritma greedy dalam pengambilan keputusan secara lokal untuk memperoleh solusi yang efisien.

### Praktikum 5 - Dynamic Programming
Implementasi dynamic programming untuk menyelesaikan masalah optimasi dengan memanfaatkan hasil perhitungan sebelumnya.

### Praktikum 6 - Graph Algorithm
Penerapan algoritma pada struktur data graf untuk pencarian jalur dan representasi hubungan antar simpul.

### Praktikum 7 - Backtracking
Implementasi algoritma backtracking untuk mencari solusi dengan mencoba berbagai kemungkinan dan melakukan proses mundur (backtrack) ketika solusi tidak valid.

**Studi Kasus:**
- Optimasi pemuatan barang pada kendaraan logistik (Container Loading Problem)

### Praktikum 8 - Branch and Bound
Implementasi algoritma Branch and Bound untuk mencari solusi optimal dengan melakukan pemangkasan (pruning) pada cabang yang tidak menjanjikan.

**Studi Kasus:**
- Optimasi rute distribusi barang (Travelling Salesman Problem)

### Praktikum 9 - A* (A-Star)
Implementasi algoritma pencarian jalur terpendek menggunakan fungsi evaluasi:

f(n) = g(n) + h(n)

**Studi Kasus:**
- Optimasi rute kurir pengiriman paket

### Praktikum 10 - Knuth-Morris-Pratt (KMP)
Implementasi algoritma pencocokan string menggunakan Failure Function untuk meningkatkan efisiensi pencarian pola.

**Studi Kasus:**
- Pencarian kode lokasi gudang pada Warehouse Management System (WMS)
